# Vue 3 Skills

Collection skills for Vue 3.

## Installation

```shell
npx skills add iwyfaf-ai-skills/vue-3 --skill vue
```

or update:

```shell
npx skills update vue
```

hard update:

```shell
npx skills add iwyfaf-ai-skills/vue-3 --skill vue
```
